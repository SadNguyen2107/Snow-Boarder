# Snow-Boarder

My 2nd Unity Game

A Snow Boarder Game

Control:
Press Left Arrow to Rotate Left
Press Right Arrow to Rotate Right
Press Shift to Go Faster

Goal:
  Get to the Finish Line

And Don't get knockout!
